# Lemosang Consulting – Website Project

This repository contains the source code for the Lemosang Consulting website.

---

## 🚀 Project Setup

### 1. Clone the Repository

```bash
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
````

### 2. Install Dependencies

Ensure you have **Node.js** and **npm** installed. You can install Node.js via [nvm](https://github.com/nvm-sh/nvm#installing-and-updating):

```bash
npm install
```

### 3. Start the Development Server

```bash
npm run dev
```

This will start a local development server with hot-reloading.

---

## 🛠 Technologies Used

* **Vite** – Fast development build tool
* **React** – JavaScript library for building user interfaces
* **TypeScript** – Strongly typed JavaScript
* **shadcn-ui** – Modern component library
* **Tailwind CSS** – Utility-first CSS framework

## ✏️ Editing the Code

You can update the project using any of the following methods:

### ✅ Using a Local Editor

1. Open the project in your IDE (e.g., VS Code).
2. Make changes.
3. Commit and push.

### ✅ Editing on GitHub

1. Browse to the file you want to change.
2. Click the **Edit** button (✏️).
3. Commit the changes to your branch or open a pull request.

### ✅ Using GitHub Codespaces (Optional)

1. Go to the repository on GitHub.
2. Click the green **Code** button and select the **Codespaces** tab.

## 🌐 Deployment

You can deploy the site using any static hosting service (e.g., Vercel, Netlify, GitHub Pages). Make sure to:
